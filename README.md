# Simple Zelda
This is a 2d Zelda-like game. Enjoy!

## Controls (approximate description):
- **&uarr;/&darr;/&larr;/&rarr;** - move up/down/left/right
- **Q** - change weapon
- **SPACE** - cast attack
- **LEFT CONTROL** - cast spell
